# NevUp Hackathon 2026 Workspace

This repo contains:

- `NevUp Hackathon 2026`: official challenge artifacts (`openapi`, JWT format, dataset, kickoff deck).
- `track3-system-of-engagement`: Track 3 production frontend (Next.js 15 + TypeScript + Tailwind + Framer Motion + TanStack Query + Zustand).

## Run From Workspace Root

```bash
docker compose up --build
```

Open:

```text
http://localhost:4173/dashboard
```

For app-level details, see:

- `track3-system-of-engagement/README.md`
- `track3-system-of-engagement/DECISIONS.md`
